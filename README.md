## 💕 자기 소개

- Backend 1년: Python/Django 기반으로 API 설계·구현, DB 모델링, 인증/권한, 서비스 로직 개발을 수행해왔습니다.

- Full-Stack 1년: (백엔드 경험과 중첩되는 기간 포함) 기획–구현–배포까지 제품 흐름을 기준으로 FE/BE를 함께 설계하며, 기능 완성도와 사용자 경험을 함께 개선해왔습니다.

강점으로는 이렇게 생각 하고 있습니다.

1. 창의력: 사용자 문제를 기능이 아닌 흐름과 경험 관점에서 재정의하고, 더 단순하고 효과적인 구조로 풀어냅니다.

2. 문제 해결력: 재현 → 원인 분석 → 대안 비교 → 개선/회고까지 근거 기반으로 빠르게 해결합니다.

3. 소통(대인관계): 요구사항을 명확히 정리하고 팀과 공유하며, 개발 과정에서 발생하는 리스크를 줄이는 협업 중심 커뮤니케이션을 중요하게 생각합니다.

제 GitHub에서 프로젝트와 코드를 확인해보세요. 감사합니다.
